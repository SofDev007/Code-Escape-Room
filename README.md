# рҹ•өпёҸвҖҚвҷӮпёҸ CODE ESCAPE ROOM v2.0
**A Matrix-Themed Cybersecurity Quiz Platform**

Code Escape Room is a high-stakes, interactive programming challenge. Players are "trapped" in a digital system and must solve coding puzzles across 7 different languages to breach vaults and escape. Powered by **NVIDIA NIM AI** for dynamic hints and performance analysis.

---

## рҹҡҖ Features
- **Hacker Aesthetic**: Immersive Matrix-rain background and terminal-style UI.
- **7 Programming Realms**: Challenges in C, C++, Java, SQL, Python, DSA, and DAA.
- **AI-Powered Hints**: Real-time coding assistance using the **Llama 3.1 8B** model.
- **Global Leaderboard**: Track performance across batches with real-time scoring.
- **AI Mission Debrief**: Personalized study advice generated by AI after every attempt.
- **Admin Dashboard**: Full control over student access, room management, and AI question generation.

---

## рҹӣ пёҸ Tech Stack
- **Backend**: Flask (Python)
- **Database**: SQLite (SQLAlchemy ORM)
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), JavaScript
- **AI Engine**: NVIDIA NIM (Meta Llama 3.1)
- **Auth**: JWT (JSON Web Tokens)

## рҹ“Ұ Local Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/code-escape-room.git
   cd code-escape-room
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Initialize the Database**
   ```bash
   python init_db.py
   ```

4. **Environment Variables**
   Create a `.env` file or set your environment variables:
   - `NVIDIA_API_KEY`: Your NVIDIA NIM API key.
   - `SECRET_KEY`: A random string for JWT security.

5. **Run the Server**
   ```bash
   python app.py
   ```
   *Access the app at `http://localhost:5000`*

## вҳҒпёҸ Deployment on Render.com

Render is the recommended platform for hosting this project.

1. **Connect GitHub**: Log in to Render and create a new **Web Service**. Connect this repository.
2. **Settings**:
   - **Environment**: `Python`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn "app:create_app()"`
3. **Add Environment Variables**:
   In the Render dashboard under **Environment**, add:
   - `NVIDIA_API_KEY`: (Your Key)
   - `FLASK_ENV`: `production`
4. **Persistent Disk (Important)**:
   Since this project uses SQLite, add a **Disk** in Render settings (Path: `/data`) and update your `config.py` to point the database to `/data/app.db`. This ensures your leaderboard and users aren't deleted on every deploy.
